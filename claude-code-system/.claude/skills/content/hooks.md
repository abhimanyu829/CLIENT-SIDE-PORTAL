# Skill: Content Hooks & Engagement

## Landing Page Hook Prompt
```
Write a hero section for our SaaS marketplace.
Target user: [PERSONA]
Their biggest pain: [PAIN POINT]
Our unique solution: [DIFFERENTIATOR]

Deliver:
1. Headline: outcome-focused, not feature-focused (10 words max)
2. Sub-headline: who it's for + how it works (1 sentence)
3. Social proof line: "[X] teams / [X] products / $[X] in transactions"
4. Primary CTA: action verb + value (e.g. "Start free trial")
5. Secondary CTA: lower commitment (e.g. "See a demo")

Test: if you removed the company name, would this headline work for a competitor?
If yes, make it more specific.
```

## Retention Email Hook Prompt
```
Write a re-engagement email for users who haven't logged in for 14 days.

Do:
- Reference something specific they left behind (e.g. "Your 3 projects are waiting")
- Make the CTA frictionless (one click back to where they were)
- Create mild urgency without lying (e.g. "Your trial ends in 7 days")

Don't:
- Use "We miss you" (overused, insincere)
- List all features
- Have more than one CTA

Subject, preview text, body (3 sentences max), CTA button.
```
