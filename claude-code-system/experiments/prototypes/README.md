# Prototypes

## Purpose
Spike code, proof-of-concepts, and early explorations.
Nothing here is production-ready or reviewed.

## Rules
1. Every prototype lives in its own subdirectory: `experiments/prototypes/[name]/`
2. Include a `README.md` explaining: what you were exploring and what you learned
3. Include outcome in `experiments/` root: `[name]-outcome.md`
4. Never import from `experiments/` into `src/` — prototype code stays here
